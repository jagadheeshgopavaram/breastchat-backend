PDF Documents Folder

This folder is where you should place your PDF files for the chatbot to process.

To get started:

1. Add your PDF files to this folder
2. Run: python vector_setup.py
3. Start the chatbot: python app.py

Example PDF files you can add:
- Research papers
- Manuals and guides
- Reports and documents
- Any text-based PDF content

The chatbot will extract text from these PDFs, create vector embeddings, and use them to answer your questions.

Note: Only PDF files (.pdf extension) will be processed.
